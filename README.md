## Features
- SSH hardening checks
- Firewall verification
- SELinux status validation
- auditd verification
- Password policy checks
- File permission checks
- Package update checks
- HTML compliance report generation
- Compliance scoring (0–100)

## Technologies
- Bash
- Linux
- systemd
- auditd
- firewalld / UFW
- HTML/CSS

## Usage
```bash
sudo ./main.sh
```
